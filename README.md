# RoboRacer ICRA 2026

Solo entry in the 5th RoboRacer Sim Racing League at ICRA 2026, Vienna.

## Algorithm Stack
- Layer 1: Gap Follower (reactive, qualification baseline)
- Layer 2: Pure Pursuit + SLAM (map-based racing)
- Layer 3: Raceline Optimization + Speed Profiling (competitive target)
- Layer 4: RL/IL Policy (stretch goal)

## Competition
- Qualification: 10 clean laps on Porto practice track
- Final: Time-attack on unseen competition track
